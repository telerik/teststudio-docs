# Search Test Steps

To search the test steps in a test by description, use the search bar.

![search](/img/features/test-maintenance/search-test-steps/search-bar.png)

Test steps are filtered as you type.

![filter](/img/features/test-maintenance/search-test-steps/type-search-bar-filter.png)

To search by test step type, select the step type menu.

![menu](/img/features/test-maintenance/search-test-steps/drop-down-filter.png)

Only selected step types appear in the test pane.

![pane](/img/features/test-maintenance/search-test-steps/filter-checked.png)

Both search filters can apply at one time.

![both](/img/features/test-maintenance/search-test-steps/both-filters-and-search.png)
