# Steps Pane Overview

![test](/img/features/test-maintenance/steps-pane/steps-pane-overview.png)

The Steps pane lists the steps in order of execution for the current test. Many actions and options are available here:

- ![3](/img/features/test-maintenance/steps-pane/step-down.png) - **Move Step Down** - Move the selected step(s) down one in the sequence.

- ![4](/img/features/test-maintenance/steps-pane/step-up.png) - **Move Step Up** - Move the selected step(s) up one in the sequence.

- ![5](/img/features/test-maintenance/steps-pane/undo-icon.png) - **Undo** - Restore a step you just deleted or changed to its previous state and location.

- ![6](/img/features/test-maintenance/steps-pane/redo-icon.png) - **Redo** - Reapply the edit that was undone with the Undo button

- ![7](/img/features/test-maintenance/steps-pane/code-behind-icon.png) - **View Code Behind File** - Coded Step only: switch the view between the step's code and the entire code behind file (if it exists).

- ![8](/img/features/test-maintenance/steps-pane/step-marker.png) - **Step Marker** - Set the step marker to denote where to insert newly recorded/added steps. 

- ![9](/img/features/test-maintenance/steps-pane/step-order.png) - **Step Order** - The step's place in the test sequence.

- ![10](/img/features/test-maintenance/steps-pane/step-checkbox.png) - **Enabled Checkbox** - Whether the step will run during execution.

- ![11](/img/features/test-maintenance/steps-pane/breakpoint-dot.png) - **Breakpoint** -  	Indicates whether the test will pause at that step and show Debug Options. (Not respected within a Test as Step. Use the Execution > Pause Test Step Property instead.)

- ![12](/img/features/test-maintenance/steps-pane/continue-on-failure.png) - **Continue on Failure** - Indicates whether the test will stop if that step detects a failure.

- ![13](/img/features/test-maintenance/steps-pane/action-icon.png) - **Step Type** - Represents the type of step: Action, Verification, Coded, etc.

- ![14](/img/features/test-maintenance/steps-pane/description-example.png) - **Description** - The default description given to the step by Test Studio.

- ![16](/img/features/test-maintenance/steps-pane/shown-inloine-editor.png) - **Show Inline Editor** - Edit some of the step properties.

- ![15](/img/features/test-maintenance/steps-pane/edit-icon.png) - **Edit Custom Description** - Alter the default step description.

- ![18](/img/features/test-maintenance/steps-pane/delete-icon.png) - **Delete** - Removes the step from the test.

- ![19](/img/features/test-maintenance/steps-pane/steps-filter-drob-down.png) - Step Type Filter - <a href="/features/test-maintenance/search-test-steps" target="_blank">Filter test steps</a> by type.
