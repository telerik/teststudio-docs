# Search Project Tests

To search the tests in a project by test name, use the project view search bar.

![search bar](/img/features/project-explorer/search-and-filter-tests/searchbar-projects.png)

Project tests filter as you type.

![filter](/img/features/project-explorer/search-and-filter-tests/type-search-filter.png)

## Filter Project Tests By Type

To filter by test type, use the test type menu.

![test type](/img/features/project-explorer/search-and-filter-tests/allsteps-filter.png)

Only selected test types appear in the project files pane.

![files pane](/img/features/project-explorer/search-and-filter-tests/allsteps-filter-checkbox.png)

Both search and filter can apply at one time.

![both](/img/features/project-explorer/search-and-filter-tests/both-search-allsteps-filter.png)

## Filter Project Tests By Usage From Other Tests

Click the <a href="/features/custom-steps/test-as-step" target="_blank">Test as Step</a> button to show all tests used as TestAsStep from another test in project.

![Test as step button](/img/features/project-explorer/search-and-filter-tests/filter-by-usage-from-other-tests-button.png)

Right click on any test used by another test offers context menu and a **'Used By'** option that shows a dialog box with listed all tests that use the initial one. 

![Context Menu->Used By ](/img/features/project-explorer/search-and-filter-tests/used-by-right-click-menu.png)

![List of Tests](/img/features/project-explorer/search-and-filter-tests/test-step-selector.jpg)

Both search and filter can apply at one time.

![Both_AsStep](/img/features/project-explorer/search-and-filter-tests/both-usage-search-filters.png)

## Filter Project Tests By Status

Click the <a href="/features/test-maintenance/tests-in-development" target="_blank">In Development</a> button to show all tests marked with 'In Development' flag to be excluded from compilation.

![In Dev button](/img/features/project-explorer/search-and-filter-tests/test-in-dev-filter-button.png)

Both search and filter can apply at one time.

![Both_InDev](/img/features/project-explorer/search-and-filter-tests/both-in-dev-search-filters.png)

The above two filters could also be applied in the same time. On top of these the search field could also be used. 

![Both_filters](/img/features/project-explorer/search-and-filter-tests/both-in-dev-usage-filters.png)
