# Export Performance Results

To export performance results:

1. Create a Performance <a href="/getting-started/test-execution/test-lists-standalone" target="_blank">Test List</a>.

![Performance Test List](/img/features/testing-types/performance-testing/export-results/fig1.png)

2. Add one or more Web Tests to the Test list.

3. Click **Run List**.

4. In the **Results** tab, double-click the test result for this test run.

![Double Click Test List](/img/features/testing-types/performance-testing/export-results/fig2.png)

5. In the **TestResults** pane, click the **View Profiler Results** icon.

![View Profiler Icon](/img/features/testing-types/performance-testing/export-results/fig3.png)

6. In the **Export** section of the Ribbon Bar, click the button for the export format of your results.

![Export](/img/features/testing-types/performance-testing/export-results/fig4.png)
